
ARG-PARSE
=====

The ArgParse aims to be a simple argument parser to `c++11`.

## Getting Started

Check out the source code
```
git clone https://github.com/szledan/arg-parse.git
```
Build `arg-parse` dynamic library
```
make arg-parse
```

## For developers

### Build & run tests

Build all tests
```
make build-tests
```
Run all tests
```
./build/bin/tests
```
